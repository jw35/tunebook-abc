Tunebook ABC

This site contains a transcription into ABC notation of the tunes used by Tuneworks in support of the workshops and sessions that they run at a number of UK folk festivals. While most of these tunes are already available on sites such as The Session, the ones here are intended to be note-for-note identical to the versions in the Tuneworks Tunebook.

  • All the tunes as a single file, suitable for further processing or importing into other programs. iPhone and iPad users might want to try out Tunebook which seems to handle this file OK.
  • One file per tune. Here (and elsewhere) the numbers on the start of the filename encode the number of the page in the original Tunebook on which the tune appears, and the tune's place on that page.

This site also contains the results of processing the ABC files into various oher formats:

  • tunebook.pdf A printable copy of the tunes using much the same layout as the Tuneworks original.
  • The printed tunebook for transposing instruments:
    • tunebook-eflat.pdf to E♭ (e.g. for alto sax)
    • tunebook-bflat.pdf to B♭ (e.g. for clarinet, soprano or tenor sax )
  • tunebook-base.pdf The printed tunebook in the base clef
  • One tune per page for various instruments:
    • tunebook-tabs.pdf with guitar chord diagrams and D whistle tabs.
    • tunebook-mandolin.pdf with mandolin tabs.
    • tunebook-dulcimer.pdf [EXPERIMENTAL] with example chord diagrams tabs of melody for D-A-D dulcimers.
    • tunebook-ukulele.pdf with ukulele chord diagrams.
  • A compact index to all the tunes, just showing the first couple of bars (handy when trying to remember the notes for a particular tune or identify a tune from its start):
    • cheatsheet.pdf just the tunes.
    • cheatsheet-whistle.pdf as above, but including D whistle tabs.
    • cheatsheet-mandolin.pdf as above, but including mandolin tabs.
    • cheatsheet-dulcimer.pdf as above, but including dulcimer tabs.
  • All the tunes converted to MIDI files
  • All the tunes converted to mp3 files

This work was partly inspired by Paul Hardy's Tunebooks and his tunebook processs page heavily informed the processing pipeline.

The master copies of these tunes, along with the tools used to produce everything else, can be found on GitHub at https://github.com/jw35/tunebook-abc.

The transcription is bound to contain mistakes, defined as (almost) anything that isn't the same as it is in the Tuneworks Tunebook (ignoring minor stylistic differences and the differences listed below). Please report any mistakes that you find, either by creating a GitHib issue or (better) a pull request, or (if that doesn't mean anything to you) by emailing Jon Warbrick <jon.warbrick@googlemail.com>.

The following differences from Tuneworks Tunebook are expected:

  • Each tune stands on its own, rather than being part of a set. As a result, they all include time signatures, and the key change warning at the end of 'The Banshee' isn't included.
  • What would be a final 'e' at the end of the first section repeat in My Darking Asleep is omitted, largely because abc2midi complains that it makes the repeat bar too long.
